# ReadingList


## INTRO
블로그, 다른 개발자들의 코드 등을 구글링하며 얻게 되는 지식도 많지만, 결국 돌고 돌아 **책에 가장 정확하고 기본이 되는 정보가 있다**고 생각됩니다.  
좋은 개발자로 성장하기 위해 읽고 싶은 책을 적어두고 읽고 있습니다.  
기본을 익혀 글로 간단히 정리하고 코드로 구현하고자 합니다.  

도서 리스트는 수시로 업데이트하며, 읽은 책에 대해서는 간단히 글 또는 코드로 정리합니다.
